Hogy kapcsolódik a Studio-hoz?

A web elküldi a promptot a local API-nak, a plugin pedig lekéri a várakozó scriptet.

Kell hozzá local szerver?

Igen, a server mappában futó Node.js bridge köti össze a webet és Studio-t.

Biztonságos?

A plugin csak új scriptet hoz létre ServerScriptService-ben, nem töröl és nem ír át meglévő fájlokat.
